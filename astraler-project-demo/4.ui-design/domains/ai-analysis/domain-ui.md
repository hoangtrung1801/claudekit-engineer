# Domain UI Design: AI Analysis Reports

> [!NOTE]
> Based on **TailAdmin** template components.
> **Focus**: Detailed, read-heavy reports.

## 1. Overview
While the Dashboard provides summaries, these pages offer deep-dive analysis reports generated by the AI pipelines (Market Landscape, Pain Points).

## 2. Page: Market Landscape Report (`/projects/{id}/analysis/market`)

### 2.1. Layout
*   **Header Action**: "Refresh Analysis" (Secondary Button, Icon: Refresh, triggers Worker).

### 2.2. Components

#### A. SWOT Analysis Matrix (Custom Component)
*   **Structure**: 2x2 Grid.
*   **Quadrants**:
    *   **Strengths** (Green Border): List of bullet points.
    *   **Weaknesses** (Red Border): List of bullet points.
    *   **Opportunities** (Blue Border): List of bullet points.
    *   **Threats** (Amber Border): List of bullet points.
*   **Style**: Each quadrant is a `Card` with a colored top border.

#### B. Competitor Positioning Map (Component: `ChartFour` - Scatter/Bubble)
*   **X-Axis**: Content Quality.
*   **Y-Axis**: Engagement Rate.
*   **Tooltip**: Competitor Name + Details.

## 3. Page: Pain Points Analysis (`/projects/{id}/analysis/pain-points`)

### 3.1. Components

#### A. Pain Point Heatmap (Component: `MapOne` or Custom List)
*   **Style**: Ranked list of "Frustrations".
*   **Row**:
    *   `Topic`: "Pricing too high".
    *   `Intensity`: 8/10 (Red Progress Bar).
    *   `Frequency`: "Mentioned in 45 comments".

#### B. Verbatim Explorer (Component: `ChatCard`)
*   **Purpose**: Show direct quotes supporting the pain points.
*   **Layout**: Scrollable list of comment snippets.
*   **Style**: "Quote" icon, italicized text, source link.

## 4. Interactions

*   **Export**: "Export PDF" button on top right of reports.
*   **Drill-down**: Clicking a SWOT bullet point opens a side-drawer with evidence (Video links/Comments).
