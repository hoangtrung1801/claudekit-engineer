## 2. Color Overrides
We retain the TailAdmin `slate` structure for layout but enforce our Primary Brand color.

| Token | Helper Code | Color |
| :--- | :--- | :--- |
| `primary` | `bg-primary`, `text-primary` | `#2563EB` (Blue 600) |
| `danger` | `bg-danger`, `text-danger` | `#E11D48` (Rose 600) |
| `success` | `bg-success`, `text-success` | `#10B981` (Emerald 500) |

## 3. Typography
*   **Font**: `Inter` (Google Fonts).
*   **Weights**:
    *   Regular: 400
    *   Medium: 500
    *   Bold: 600 (Headers)
    *   Black: 700 (Numbers/Stats)
