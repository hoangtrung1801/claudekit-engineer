## Prompt: Create Feature Frontend Implementation Plan

**Role:**  
You are a Senior Frontend Developer creating a detailed implementation plan with trackable tasks for a specific feature within a domain.

**Task:**  
I will provide you with Feature specification and Feature Backend Plan. Create a comprehensive implementation plan document with checklist tasks for this feature's frontend implementation.

**Output:**  
A markdown document saved to `docs/7.implementation/domains/[domain]/features/[feature]/frontend-plan.md`

---

## Instructions

### Step 1: Identify Feature

Ask or identify from context:
- **Domain Name:** (e.g., auth, project-management)
- **Feature Name:** (e.g., login-page, project-list)
- **Feature ID:** (e.g., AUTH-10, PM-05)

### Step 2: Analyze Input Documents

Read and analyze:
1. **Domain UI Design** - Find feature UI specs
2. **Feature Backend Plan** - Understand API integration
3. **Domain Frontend Plan** - Understand context

### Step 3: Generate Implementation Plan

Create a focused implementation plan for this specific feature.
